# Suggested Screenshot Shot List

Use this as a baseline list for the manual.

## Core Screens

- Main window (empty state)
- Main window (configured with several sessions)
- Chat window with `/help` output
- Edit View dialog with HA sensor mapping visible
- Edit Control dialog with Active Folder path configured
- Settings window with `Validate config`/`Import`/`Export`
- Position/Layout tool in `Position` mode
- Position/Layout tool in `Session` mode

## Operational Flows

- Open one View session
- Open one Control session
- Tagged open/close flow
- Setup save/apply flow
- Linked session open flow
- Takeover flow with chat notice
- Reconnect-on-drop enabled state

## Home Assistant Examples

- Sensor icon row with multiple indicators
- Binary true/false icon behavior example
- Notification state example (alarm icon)

## Capture Guidelines

- Keep same window size between related screenshots.
- Use a neutral desktop background.
- Hide sensitive hostnames/IPs/tokens.
- Prefer PNG output.
- Keep text zoom at 100% unless documenting accessibility mode.
